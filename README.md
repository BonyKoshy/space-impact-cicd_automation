# 🚀 Space Impact Fork!

A retro-style 2D space shooter built using **HTML5 Canvas, CSS, and JavaScript**. Inspired by classic arcade game (Nokia space-impact), this project delivers fast-paced gameplay with smooth animations, enemy waves, and power-ups.

---

## 🎮 Features

- 🚀 Player-controlled spaceship with smooth movement
- 👾 Multiple enemy types with unique behaviors
- 🔫 Shooting mechanics with upgrades (Rapid Fire, Triple Shot)
- 🛡️ Power-ups: Shield, Extra Life, Weapon Boosts
- 💥 Particle explosion effects
- 🌌 Animated starfield background
- 📊 HUD displaying Score, Level, and Lives
- 📱 Mobile-friendly controls (touch support)
- ⌨️ Keyboard controls for desktop
- 🔁 Game states: Menu, Playing, Dead, Game Over

---

## 🕹️ Controls

### Desktop

- ⬆️⬇️⬅️➡️ Arrow Keys – Move
- 🔫 `Space` – Shoot / Start Game

### Mobile

- On-screen D-pad – Move
- FIRE button – Shoot / Start

---

## ⚙️ Technologies Used

- HTML5 Canvas
- CSS3 (Retro UI styling)
- ES6 Modules (Vanilla JavaScript)
- Vite (Build Tool & Dev Server)

---

## 🧠 Concepts Covered

- Game loop using `requestAnimationFrame`
- Collision detection
- State management
- Event handling (keyboard & touch)
- Object-based game architecture
- Animations & particle systems

## 🚀 Getting Started

1. **Clone the repository:**

   ```bash
   git clone https://github.com/your-username/space-impact.git
   cd space-impact
   ```

2. **Install dependencies:**

   ```bash
   npm install
   ```

3. **Start the local development server:**

   ```bash
   npm run dev
   ```

4. **Build for production:**
   ```bash
   npm run build
   ```
